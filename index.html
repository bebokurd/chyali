Is Cydia Elite free to use?

Absolutely. Cydia Elite is a community-driven project. All integrated tools, package links, and search utilities are provided 100% free of charge with no hidden subscriptions.

How do I install packages?

Simply find the package you want. If it has a URL, clicking it will take you to the official download page. If it has a command, clicking will copy it to your clipboard for use in PowerShell or Terminal.

Does it work on mobile?

Yes! Cydia Elite is fully responsive and designed to work seamlessly on iOS, Android, and desktop browsers. Some terminal tools may require a PC, but all streaming and search features work everywhere.

Can I request a new tool?

We love community feedback! You can join our Discord server or message Chya on Instagram to suggest new tools, scripts, or streaming sources to be added to the portal.

How do I use the Developer API Hub?

The Developer API Hub is a sandboxed playground. In the 'Documentation' tab, you can click glowing 'Try Now' badges next to routes to instantly switch to the 'API Tester' tab and pre-fill details. The 'API Tester' lets you configure HTTP methods (GET, POST, PUT, DELETE), dynamically build key-value HTTP headers, add raw JSON request payloads, and visualize execution metrics.

What is the secureFetch network engine?

Our centralized secureFetch engine ensures network request safety. It uses AbortController to enforce a strict 10-second request timeout limit. Additionally, if a direct REST query gets blocked due to CORS browser policies, secureFetch automatically fallbacks and routes the request securely via the AllOrigins proxy, reconstructs a valid Response, and resolves it flawlessly.

How do I use custom URL hash routing?

URL hash routing preserves filter states. Clicking category links creates paths like #installed/social. You can also visit clean, bookmarked aliases directly (e.g. #community, #live-sports, or #ad-blockers) and the portal router will immediately parse, filter, and synchronize visual sidebar highlights automatically.

How does the Admin Dashboard pagination work?

Navigating to /admin loads our database management dashboard. It uses a high-performance pagination bar supporting page sizes (10, 15, 25, 50, 100). Selecting 'All' disables bounds, listing all 220+ entries simultaneously in the DOM table, which simplifies sweeping edits.

Are my searches and downloads private?

Privacy is our core pillar. We do not log your search queries, IP addresses, or download history. All tool processing happens in real-time and data is cleared immediately after your session.

How often is the database updated?

Our package database (data.json) is updated frequently. New tools, streaming links, and security patches are usually added weekly to ensure everything remains operational.
